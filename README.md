# Лабораторная работа №1 M3

## Калькулятор
Этот проект реализует вычисление математических выражений, записанных в постфиксной польской нотации. Он поддерживает основные арифметические операции, унарные операторы, а также скобки для группировки выражений внутри постфиксных выражений.


## Функционал

- **Обработка чисел** (целых и дробных) — ввод может содержать числа вроде `3`, `4.5` и т.д.
- **Арифметические операции**:
  - `+` — сложение
  - `-` — вычитание
  - `*` — умножение
  - `/` — деление (с обработкой деления на ноль)
  - `**` — возведение в степень
  - `%` — взятие остатка (только для целых чисел)
  - `//` — целочисленное деление (только для целых чисел) (с обработкой деления на ноль)
- **Унарные операторы**:
  - `~` — унарный минус
  - `$` — унарный плюс
- **Скобки**:
  - `(` и `)` позволяют вычислять выражения внутри скобок, что помогает структурировать сложные выражения.

## Обработка ошибок
**При некорректных выражениях программа выдаст сообщение об ошибке:**

- Некорректное подвыражение
- Недостаточно символов для операции
- Деление на ноль
- Операции `//` и `%` с нецелыми числами
- Неверная запись с унарными операторами
- Некорректный символ в записи

## Примеры
<pre>
Введите выражение в постфиксной польской нотации: ( 8 ( 3 2 + ) - ) 4 *
Результат: 12.0
</pre>

<pre>
Введите выражение в постфиксной польской нотации: ( 3 4 + ) ( 5 2 - ) *
Результат: 21.0
</pre>
